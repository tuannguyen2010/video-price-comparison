# video-price-comparison


## APPLICATION SETUP
1. Back end package install
   
   ```
   cd back-end
   npm install
    ```
2. Run back-end
   ```
   npm run start
    ```

3. Open different terminal
       
4. Front end package install
   ```
   cd front-end
   npm install
   ```
   
5. Run front-end
    ```
   npm run start
    ```
